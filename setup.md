# Qovyn Studio Setup Guide

## 1. Download & Install

1. Go to [https://qovyn-studio.vercel.app/](https://qovyn-studio.vercel.app/) and download the setup.
2. Run the installer and follow the installation steps.
3. Approve any security prompts during installation.

---

## 2. Configure AI Providers

1. Launch Qovyn Studio.
2. Click the **Settings** button in the bottom-left corner.
3. Navigate to the **AI Providers** tab.
4. Each provider has a **"Get key"** link to obtain an API key.
5. Add a model ID for each provider to enable model selection.

### Recommended Providers

| Provider | Description |
|----------|-------------|
| Puter JS | Free tier, fast setup |
| Qovyn Router (OpenRouter) | Multiple models via one API |
| Open Code Zen | Code-optimized models |
| Z AI | General purpose AI |
| NVIDIA NIM | NVIDIA-hosted models |
| Mistral | Open-weight models |

---

## 3. Quick Setup: Get a Puter API Key

1. Sign in at [https://puter.com/](https://puter.com/).
2. Navigate to your dashboard — either:
   - Click **Account** → **Settings**, or
   - Open [https://puter.com/dashboard](https://puter.com/dashboard) in a new tab.
3. In the dashboard, click **Account** in the sidebar.
4. Click **Create Token**.
5. Give the token a name and click **Create Token**.
6. Copy the generated token.
7. In Qovyn Studio, paste it into the Puter provider's **API Key** field.
8. Click **Save** and **Verify Token**.
9. Once verified, Puter models will appear in the AI panel model selector.

You can now use any model from the configured provider in the AI panel.

---

## 4. Login for Full Features

For better agent performance (web searches, advanced tools, etc.), log in to the app:

1. Click the **user icon** in the top-right corner to open the login modal.
2. Click **Continue to Web Login**.
3. On the web login page, sign in with your **Google** or **GitHub** account.
4. You will be taken to your dashboard.
5. Click **Launch Qovyn** — the app will log you in automatically.

### If Login Fails

1. Scroll to the bottom of the dashboard page.
2. Click **Copy Session Token**.
3. In the app's login modal, click **"Problem with redirect? Enter manual token"**.
4. Paste the token and click to log in.

---

## 5. Usage & Limits

- **Free tier:** Up to **15 requests per day**.
  - A single request can generate a complete app, website, or project.
- **Unlimited plan:** Upgrade for **$6 / 3500 FCFA per month** via the website's pricing section.

Once logged in and configured, you're ready to start coding.

# Guide d'Installation de Qovyn Studio

## 1. Téléchargement & Installation

1. Allez sur [https://qovyn-studio.vercel.app/](https://qovyn-studio.vercel.app/) et téléchargez le programme d'installation.
2. Exécutez l'installateur et suivez les étapes d'installation.
3. Approuvez les invites de sécurité pendant l'installation.

---

## 2. Configurer les Fournisseurs IA

1. Lancez Qovyn Studio.
2. Cliquez sur le bouton **Paramètres** dans le coin inférieur gauche.
3. Accédez à l'onglet **Fournisseurs IA**.
4. Chaque fournisseur a un lien **"Obtenir une clé"** pour obtenir une clé API.
5. Ajoutez un ID de modèle pour chaque fournisseur pour activer la sélection de modèles.

### Fournisseurs Recommandés

| Fournisseur | Description |
|-------------|-------------|
| Puter JS | Niveau gratuit, configuration rapide |
| Qovyn Router (OpenRouter) | Plusieurs modèles via une seule API |
| Open Code Zen | Modèles optimisés pour le code |
| Z AI | IA à usage général |
| NVIDIA NIM | Modèles hébergés par NVIDIA |
| Mistral | Modèles à poids ouverts |

---

## 3. Configuration Rapide : Obtenir une Clé API Puter

1. Connectez-vous sur [https://puter.com/](https://puter.com/).
2. Accédez à votre tableau de bord — soit :
   - Cliquez sur **Compte** → **Paramètres**, ou
   - Ouvrez [https://puter.com/dashboard](https://puter.com/dashboard) dans un nouvel onglet.
3. Dans le tableau de bord, cliquez sur **Compte** dans la barre latérale.
4. Cliquez sur **Créer un Jeton**.
5. Donnez un nom au jeton et cliquez sur **Créer un Jeton**.
6. Copiez le jeton généré.
7. Dans Qovyn Studio, collez-le dans le champ **Clé API** du fournisseur Puter.
8. Cliquez sur **Enregistrer** et **Vérifier le Jeton**.
9. Une fois vérifié, les modèles Puter apparaîtront dans le sélecteur de modèles du panneau IA.

Vous pouvez maintenant utiliser n'importe quel modèle du fournisseur configuré dans le panneau IA.

---

## 4. Connexion pour les Fonctionnalités Complètes

Pour de meilleures performances de l'agent (recherches web, outils avancés, etc.), connectez-vous à l'application :

1. Cliquez sur l'**icône utilisateur** dans le coin supérieur droit pour ouvrir la fenêtre de connexion.
2. Cliquez sur **Continuer vers la Connexion Web**.
3. Sur la page de connexion web, connectez-vous avec votre compte **Google** ou **GitHub**.
4. Vous serez redirigé vers votre tableau de bord.
5. Cliquez sur **Lancer Qovyn** — l'application vous connectera automatiquement.

### Si la Connexion Échoue

1. Faites défiler jusqu'en bas de la page du tableau de bord.
2. Cliquez sur **Copier le Jeton de Session**.
3. Dans la fenêtre de connexion de l'application, cliquez sur **"Problème de redirection ? Entrer le jeton manuel"**.
4. Collez le jeton et cliquez pour vous connecter.

---

## 5. Utilisation & Limites

- **Niveau gratuit :** Jusqu'à **15 requêtes par jour**.
  - Une seule requête peut générer une application, un site web ou un projet complet.
- **Plan illimité :** Passez à **$6 / 3500 FCFA par mois** via la section tarification du site Web.

Une fois connecté et configuré, vous êtes prêt à commencer à coder.
